My Projects

Image Processing with MATLAB and Partial Differential Equations (PDEs)

Click on the picture to learn more about this project.

Description: The main objective of this project was to be able to utilize specific MATLAB programming and knowledge of Partial Differential Equations (PDEs) towards denoising images that have been corrupted by Gaussian noise. The primary PDE that was focused for this project was the diffusion/heat PDE, ut = kuxx, such that k is the diffusion coefficient (this was set to equal 1 throughout the project), with Neumann and Dirichlet boundary conditions, where these boundary conditions become specialized image denoising conditions when a general solution for the PDE is found at some t > 0 time.

  • With Dr. Jeungeun Park of SUNY New Paltz
  • Original image obtained from Hanayo Sora (@SoraHanayo) from Twitter
  • Project signed and approved in December 2021 by SUNY New Paltz Dept. Chairperson, Dr. David Hobby
